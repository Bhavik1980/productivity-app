# Personal Productivity Web App

A web application that helps users manage their daily tasks efficiently using Pomodoro technique, task categories, and dark/light mode toggle. The app features task creation, filtering, completion metrics, and a refreshing Pomodoro timer.

## Features

- **Add, delete, and complete tasks**: Easily manage tasks by adding new ones, marking them as complete, or deleting them.
- **Task categories**: Organize tasks by categories (Work, Study, Personal).
- **LocalStorage persistence**: Tasks persist even after refreshing the page.
- **Category filtering**: Filter tasks by category.
- **Completion metrics**: View total tasks, completed tasks, and pending tasks.
- **Clear completed tasks**: Easily clear completed tasks with one click.
- **Dark / Light mode toggle**: Switch between dark and light themes for a better visual experience.
- **Responsive design**: Fully responsive, works seamlessly on both desktop and mobile devices.
- **Smooth animations**: Animations when tasks are added or completed for better user experience.
- **Pomodoro Timer**: Track focused work time with the Pomodoro timer (user-defined duration). Option for work and break sessions.

## Tech Stack

- **React (Vite)**: The app is built using React for the front-end and Vite as the build tool.
- **CSS3**: For styling and responsive design.
- **LocalStorage API**: For persistent task data across browser refreshes.
